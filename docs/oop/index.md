# Objektorientierung (OOP)

Die objektorientierte Programmierung (OOP) ist ein Programmierparadigma, das auf dem Konzept von "Objekten" basiert. Diese Objekte enthalten Daten und Code: Daten in Form von Feldern (oft als Attribute bezeichnet) und Code in Form von Methoden.

Das Ziel von OOP ist es, komplexe Software einfacher zu strukturieren, wiederverwendbar zu machen und leichter zu warten.
